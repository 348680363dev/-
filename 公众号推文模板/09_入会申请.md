## 一、入会条件

**基本要求：**

| 条件 | 说明 |
|------|------|
| 年龄 | 【如：18-70周岁，身体健康】 |
| 健康状况 | 无心脏病、高血压、癫痫等禁忌疾病 |
| 体检证明 | 需提供3个月内二级甲等及以上医院体检证明 |
| 热爱冬泳 | 认同协会章程，愿意遵守协会规定 |
| 【其他】 | 【如有】 |

**以下人员暂不接受申请：**
- 心脏病、高血压等重大疾病未控制者
- 精神疾病或认知障碍患者
- 酗酒或药物依赖者
- 【其他限制】


---

## 二、入会流程

```
① 提交入会申请（点击下方表单）
       ↓
② 协会审核（1-3个工作日）
       ↓
③ 审核通过，联系入群
       ↓
④ 缴纳会费【】元/年
       ↓
⑤ 参加一次体验活动
       ↓
⑥ 正式成为会员
```

---

## 三、会费标准

| 会员类型 | 年费 | 权益 |
|----------|------|------|
| 正式会员 | 【】元/年 | 全部会员权益 |
| 体验会员 | 免费 | 可参加【】次体验活动 |
| 荣誉会员 | 免费 | 【条件，如：年满70周岁】 |

**会费用途：** 场地租赁、器材购置、活动组织、宣传费用等

---

## 四、入会必读

### 装备要求

| 装备 | 说明 | 是否必须 |
|------|------|----------|
| 泳镜 | 保护眼睛，适应水下视野 | ✅ 必选 |
| 泳帽 | 标识身份，保护头发 | ✅ 必选 |
| 泳裤/泳衣 | 专业泳衣最佳 | ✅ 必选 |
| 浴巾/毛巾 | 至少2条，上岸后擦干 | ✅ 必选 |
| 保暖衣物 | 羽绒服/抓绒衣等，上岸后立即保暖 | ✅ 必选 |
| 拖鞋 | 岸上行走使用 | ✅ 必选 |
| 防寒泳帽/手套 | 冬季建议配置 | 🟡 建议 |

### 注意事项

1. 入会需如实填写健康状况，如隐瞒病情，后果自负
2. 参加活动需服从领队安排，不得擅自行动
3. 首次参加活动建议有人陪同，让领队了解您的水平
4. 遇身体不适请立即上岸并告知领队
5. 积极参加协会活动，爱护公共设施
