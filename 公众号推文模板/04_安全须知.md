## 合肥市冬泳协会·安全须知

⚠️ **冬泳有风险，下水需谨慎！请所有会员仔细阅读以下安全准则**

---

## 一、冬泳入门禁忌

以下情况严禁下水：

| 禁忌事项 | 说明 |
|----------|------|
| 空腹或饱餐后 | 空腹血糖低，饱餐后血液集中在消化系统，均易出事 |
| 熬夜/饮酒后 | 反应迟钝，体温调节能力下降 |
| 身体不适/发烧 | 免疫力低下，不宜参加任何运动 |
| 心脏病/高血压未控制 | 高风险疾病患者需医生评估后方可参加 |
| 女性生理期 | 建议暂停，避免感染和着凉 |
| 未做热身运动 | 直接下水极易抽筋 |

---

## 二、四季下水安全准则

### 🌸 春季（下水前后）
- 水温回升但仍较低，下水前充分热身
- 上岸后立即擦干身体，换上干衣服
- 防止早晚温差大导致感冒

### ☀️ 夏季
- 避开烈日高温时段，选择清晨或傍晚
- 注意防晒，避免长时间曝晒
- 补充水分，防止中暑

### 🍂 秋季
- 水温逐渐下降，逐步增加下水时间
- 留意天气变化，大风大雨天气暂停
- 上岸后尽快保暖

### ❄️ 冬季
- 严格控制水温对应的安全时长（见下表）
- 必须结伴同行，切勿独自下水
- 携带保暖衣物、毛巾、姜茶等

**冬泳安全时长参考（水温→最大时长）：**

| 水温 | 建议最长游泳时间 |
|------|------------------|
| 10℃以上 | 30分钟以内 |
| 5-10℃ | 15-20分钟 |
| 0-5℃ | 5-10分钟 |
| 0℃以下 | 不建议下水 |

## 三、野外水域风险提示

⚠️ **协会活动以正规游泳馆和备案安全水域为主，禁止随意前往不明水域！**

### 野外水域风险清单

| 风险类型 | 说明 | 应对措施 |
|----------|------|----------|
| 水流/暗流 | 河流、湖泊可能存在不可见水流 | 禁止进入不熟悉的水域 |
| 水草/杂物 | 可能缠绕肢体 | 选择无水草区域下水 |
| 水底地形不明 | 可能有深坑、岩石 | 提前勘察或使用浮标标记 |
| 水质问题 | 污染、寄生虫风险 | 选择经检测的水域 |
| 低温 | 野外水温可能低于预期 | 必须携带温度计，严格控制时间 |
| 突发天气 | 暴雨、大风、雷电 | 收到预警立即上岸 |

## 四、急救常识

### 抽筋急救
1. 保持冷静，立即向岸边或同伴靠近
2. 手指抽筋：用力握拳，再用力伸直，反复几次
3. 小腿抽筋：用手抓住脚尖，向身体方向拉，小腿用力蹬直
4. 大腿抽筋：弯曲膝盖，双手抱住小腿，用力向身体方向拉

### 呛水急救
1. 立即侧身或仰漂，保持口鼻在水面上
2. 大声呼救，等待救援
3. 意识清醒后可划水靠岸

### 失温急救
1. 立即上岸，去除湿衣
2. 用干毛巾/衣物擦干身体
3. 置于避风处，裹以保暖衣物
4. 给予温热饮品（禁止酒精、咖啡）
5. 严重者立即送医

### 心脏骤停急救
1. 立即呼叫120
2. 进行心肺复苏（CPR）+人工呼吸
3. 如有AED设备，立即使用
