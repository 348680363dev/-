## 🚀 使用步骤

### 第一阶段：基础搭建（必须）
1. ☐ 填写01-09推文模板（走进协会+活动中心）
2. ☐ 创建腾讯文档报名表+入会申请表
3. ☐ 配置公众号底部菜单（跳转对应推文）
4. ☐ 发布首批推文（建议一次发1-2篇）

### 第二阶段：内容完善（建议）
5. ☐ 填写10-13推文模板（风采展示）
6. ☐ 创建腾讯相册，上传照片
7. ☐ 整理视频号，发布首批视频
8. ☐ 更新相册/视频集锦推文

### 第三阶段：持续运营（长期）
9. ☐ 每次活动：发预告→活动后发回顾
10. ☐ 每周更新：集训安排推文
11. ☐ 每月1-2期：会员风采+科普干货
12. ☐ 相册持续更新
